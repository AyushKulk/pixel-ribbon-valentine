# Collage decorations

Add these PNG images (with transparent backgrounds) to this folder:

| Filename | Description |
|----------|-------------|
| `black_cat.png` | Cute stylized black cat |
| `white_brown_cat.png` | Cute stylized white and brown cat |
| `yoshimoto_nara.png` | Yoshimoto Nara skateboard art style |
| `vivienne_westwood.png` | Vivienne Westwood logo |
| `maison_margiela.png` | Maison Margiela logo |
| `matcha.png` | Matcha whisk and cup |
| `nyc.png` | NYC logo |

These will appear around the Miffy & Smiski graphic at the bottom of the collage. Images are scaled to ~56px and arranged in a circle.
